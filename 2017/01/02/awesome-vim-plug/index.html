VIM插件管理器 - vim-plug

2017-01-02
Editor
约 230 字 预计阅读 1 分钟 次阅读

文章目录

      • 为什么要用
      • 怎么用

之前一直在用spf13-vim的开源vim配置，但是一直都感觉这个太过重量级了。用了大半年之后还是决定自己动手diy一个。

为什么要用

主要是因为vim-plug安装插件是并行的，比vundle要快得多，而且提示信息也更加丰富。

怎么用

安装

1
2

curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim

使用

在.vimrc文件中加入：

1
2
3
4
5
6
7

call plug#begin('~/.vim/plugged')

" Shorthand notation; fetches https://github.com/junegunn/vim-easy-align
Plug 'junegunn/vim-easy-align'

" Initialize plugin system
call plug#end()

在begin和end中间加入自己要安装的插件即可。

常用命令

  • 查看插件状态 :PlugStatus
  • 安装插件 :PlugInstall
  • 更新插件 :PlugUpdate
  • 清理插件 :PlugClean

更多用法可以参考项目地址：vim-plug
