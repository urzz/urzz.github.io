Chrome 好用的插件及介绍

Infinity new tab

自定义新建的标签页，集合了很多很有用的功能。

GitHub Awesome Complete

不离开当前项目窗口的前提下，弹窗显示出你的搜索结果

Octotree

Octotree 是一款非常方便的 GitHub 扩展，安装了这款插件之后，你可以在 GitHub 项目页面的左侧看到相应项目的树状结构图，对于项目管理以及具体的文件跳转来说非常方便。

Postman

相信开发者朋友一定知道这款插件，这是一款强大的 API & HTTP 请求调试工具，它不仅可以调试简单的 HTML、CSS 以及脚本等简单的网页基本信息，这款 Chrome 插件甚至还能发送几乎所有的 HTTP 请求

掘金浏览器插件

专为开发者、设计师和产品经理量身定制，能够帮助你快速发现优质的开发、设计或者产品方面的干货，减少筛选成本。

Proxy SwitchySharp

翻墙工具，配合Shadowsocks使用，灵活。可以设置PAC文件，以及手动添加PAC规则列表。轻松快捷地管理和切换多个代理设置。

Google 翻译

快捷的对网页上的内容进行翻译，浏览网页时可轻松查看翻译版本。由Google翻译小组提供。

Stylish

是一个调整网页外观的用户样式管理器。它可让您轻松地为许多热门网站网站安装主题和皮肤。

RSS Feed Reader

方便管理自己的RSS的订阅

uBlock Origin

一款高效的网络请求过滤工具，占用极低的内存和 CPU。用来过滤广告不错。

知乎外链直达

移除知乎外部链接重定向，加速上网、避免访问追踪。

马克飞象

强大的Markdown离线编辑器，专为印象笔记打造

Advanced Font Settings

修改浏览器的字体

一键管理所有扩展

一款可以一键管理所有 Chrome 扩展的 Chrome 扩展。

Tampermonkey

油猴脚本Tampermonkey，可以装很多实用的网站脚本
